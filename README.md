# Bootstrap Responsive Website

This project is a responsive website built using Bootstrap. It demonstrates the use of Bootstrap components, grid system, and responsive design principles to create a clean and user friendly web interface.


## Project Structure

Bootstrap/
├── bootstrap.html
├── index.html
├── logo.png
├── slide1.png
├── slide2.png
├── slide3.png
└── success.png


## Project Description

This Bootstrap project contains multiple HTML pages styled using Bootstrap components. Images such as logos and slides are used to enhance the visual appearance of the website. The project is designed as a frontend-only application and does not require any backend or database.

## Features

- Static responsive web pages
- Bootstrap based layout
- Image slider support
- Clean and simple UI
- Easy to understand structure

## Technologies Used

- HTML5
- Bootstrap
- CSS
- JavaScript (basic usage if included via Bootstrap)

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. The website will open in your default browser

No installation or server setup is required.

## Pages Included

- `index.html` – Main landing page
- `bootstrap.html` – Bootstrap components and layout page
